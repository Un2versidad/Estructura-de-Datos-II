 Diseña una aplicación que permita administrar un inventario con las siguientes características:



1\. Arreglos

Almacenar un listado inicial de productos con atributos (nombre, código, cantidad, precio).



Implementar una función para ordenar los productos alfabéticamente o por precio.



2\. Listas Enlazadas

Permitir agregar y eliminar productos del inventario de forma dinámica.



Implementar una búsqueda por código utilizando una lista enlazada.



3\. Pilas

Llevar un historial de las últimas 10 operaciones realizadas (agregar, eliminar, editar).



Permitir deshacer la última operación.



4\. Colas

Simular una fila de pedidos de clientes. Cada cliente solicita uno o varios productos.



Atender a los clientes en orden de llegada.



5\. Árbol Binario de Búsqueda (ABB)

Insertar productos en un árbol ordenado por código o nombre.



Implementar búsqueda eficiente de un producto.



Mostrar el recorrido inorden del árbol para ver los productos ordenados.
